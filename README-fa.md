# 🦢 گوس‌من (Gooseman)

گوس‌من یک داشبورد وب سبک برای مدیریت و مانیتور کردن کلاینت درحال اجرای GooseRelayVPN روی سیستم محلی یا سرور LAN است.

این پروژه یک پنل کنترل ساده برای استارت و استاپ کردن کلاینت، مشاهده لاگ‌های لحظه‌ای، دنبال کردن آمار مصرف، و ویرایش تنظیمات پراکسی SOCKS از طریق مرورگر فراهم می‌کند.

<img alt="Gooseman dashboard" src="assets/screenshot.png" style="max-width: 100%; height: auto;" />

## ویژگی‌ها

- داشبوردی زیبا، مدرن و چشم‌نواز.
- **رابط کاربری کاملاً آفلاین:** تمامی کتابخانه‌های CSS و JS (از جمله Tailwind و Chart.js) در پروژه گنجانده شده‌اند. برای بارگذاری ظاهر و نمودارها نیازی به اینترنت بین‌الملل نیست.
- پشتیبانی از باینری‌های ویندوز و لینوکس (فایل اجرایی منتشر شده مختص ویندوز است).
- امکان استارت و استاپ کردن کلاینت GooseRelayVPN از طریق رابط وب.
- نمایش زنده لاگ‌ها با بروزرسانی خودکار و اسکرول هوشمند.
- نمایش لحظه‌ای آمار سشن، ترافیک و مانیتورینگ داینامیک اسکریپت‌ها.
- **توقف هوشمند (Auto-Stop Quota):** امکان متوقف شدن خودکار سرور حین رسیدن به یک لیمیت تعیین‌شده برای جلوگیری از مسدود شدن اکانت.
- احراز هویت داشبورد با رمز عبور مستقل که خارج از فایل کانفیگ اصلی ذخیره می‌شود.
- ویرایشگر تنظیمات SOCKS5 (هاست، پورت، نام کاربری و رمز عبور اختیاری).
- طراحی واکنش‌گرا برای دسکتاپ و موبایل.
- بک‌اند سبک مبتنی بر FastAPI بدون وابستگی خارجی به‌جز پکیج‌های پایتون.
- نصب و راه‌اندازی بسیار آسان.

---

## معماری پروژه

گوس‌من از دو بخش اصلی تشکیل شده است:

- **بک‌اند (FastAPI)**
  - اجرا و مدیریت پروسه `goose-client`
  - خواندن و پردازش لاگ‌های چند-اکانتی از خروجی استاندارد (stdout)
  - دنبال کردن آمار اجرا، وضعیت اسکریپت‌ها و میزان مصرف
  - ارائه یک API ساده HTTP برای داشبورد

- **فرانت‌اند (Single-page HTML)**
  - ساخته‌شده با TailwindCSS و Chart.js (آفلاین)
  - ارتباط با بک‌اند از طریق JavaScript Fetch API
  - نمایش زنده لاگ‌ها، آمار و کنترل‌ها

## پیش‌نیازها (برای اجرای سورس‌کد)

*نکته: در صورت استفاده از فایل اجرایی (`.exe`) نیازی به نصب پایتون و پیش‌نیازها ندارید.*

- پایتون 3.9 یا بالاتر
- FastAPI
- Uvicorn
- باینری GooseRelayVPN (`goose-client` یا `goose-client.exe`) در ریشه پروژه
- فایل `client_config.json` در ریشه پروژه

## نصب و اجرا

### روش اول: نسخه اجرایی مستقل (مختص ویندوز)
۱. آخرین نسخه `Gooseman.exe` را از بخش **[Releases](../../releases)** دانلود کنید.
۲. فایل `goose-client.exe` را دقیقاً در کنار آن قرار دهید.
۳. روی `Gooseman.exe` دبل‌کلیک کنید. مرورگر شما به صورت خودکار داشبورد را باز خواهد کرد!
*(در صورت عدم وجود `client_config.json`، برنامه یک فایل خام و استاندارد برای شما می‌سازد).*

### روش دوم: از طریق سورس‌کد پایتون
۱. کلون کردن ریپازیتوری:

```bash
git clone https://github.com/Aydiniyom/Gooseman.git
cd Gooseman
```

۲. نصب وابستگی‌ها:

```bash
pip install -r requirements.txt
```

۳. فایل‌های `client_config.json` و `goose-client` خود را داخل پوشه اصلی پروژه قرار دهید.

۴. داشبورد را با دستور زیر اجرا کنید:

```bash
uvicorn main:app --host 0.0.0.0 --port 5000
```

سپس با آدرس `http://localhost:5000` به آن دسترسی خواهید داشت.
